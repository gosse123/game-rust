# 🎮 Jeu de Devinette de Nombre en Rust

Un petit jeu en ligne de commande développé en Rust où le joueur doit deviner un nombre généré aléatoirement par l’ordinateur.

---

# 🚀 Fonctionnalités

- Génération aléatoire d’un nombre secret
- Saisie utilisateur depuis le terminal
- Indications :
  - Trop grand 📈
  - Trop petit 📉
- Détection de la bonne réponse ✅
- Gestion des erreurs de saisie

---

# 🛠️ Technologies utilisées

- Rust
- Cargo
- Crate `rand`

---

# 📦 Installation

Clone le projet :

```bash
git clone https://github.com/votre-utilisateur/nom-du-projet.git
```

Entrer dans le dossier du projet :

```bash
cd nom-du-projet
```

Compiler le projet :

```bash
cargo build
```

---

# ▶️ Exécution

```bash
cargo run
```
